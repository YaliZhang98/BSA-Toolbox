Length detection

This program can find average\maximum\minimun lengths of sequences in fasta\fastq file.


Upload a fasta or fastq file with sequences (Allowed file format: fasta file - fasta\fna\fa; fastq file - fastq\fq)



Tips: Both of uppercase or lowercase nucleotide in the input file are allowed.



If you need change the input values, click the "Reset" button, the form-data will be reset to the blank.
