Translation

This program can translate nucleotide sequences in fasta\fastq file to amino acid sequence.

This translation will start from the first nucleotide in sequence. If sequence length is not a multiple of three, the nucleotides that cannot form a codon at the end of the sequence will be discarded. If a combination of three nucleotides is not a codon, this will appear as '-'. Stop codons appear as '*'
