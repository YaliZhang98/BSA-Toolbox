GC_content plot

This program can plot GC content over a sequence where the user can define window size and step.

Tips: User can input a pure nucleotide sequence or upload a fasta\fastq file. All nucleotide sequences in fasta\fastq file will be combined into one sequence and GC-content plot is result of that one sequence. Both of uppercase or lowercase nucleotide in the input text\file are allowed.

Input pure nuceotide sequence (eg. atcgatcg)



Or upload a fasta or fastq file with nucleotide sequences (Allowed file format: fasta file - fasta\fna\fa; fastq file - fastq\fq)
