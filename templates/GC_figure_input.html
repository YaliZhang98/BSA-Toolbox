GC_content plot

This program can plot GC content over a sequence where the user can define window size and step.

User can input a pure nucleotide sequence or upload a fasta\fastq file. All nucleotide sequences in fasta\fastq file will be combined into one sequence and GC-content plot is result of that one sequence.
