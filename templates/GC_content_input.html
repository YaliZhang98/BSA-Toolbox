GC content calculation

This program can calculate GC content of nuceotide sequence.

Input pure nuceotide sequence (eg. atcgatcg)



Or upload a fasta or fastq file with nucleotide sequences (Allowed file format: fasta file - fasta\fna\fa; fastq file - fastq\fq)



Tips: Both of uppercase or lowercase nucleotide in the input text\file are allowed.




If you need change the input values, click the "Reset" button, the form-data will be reset to the blank.
